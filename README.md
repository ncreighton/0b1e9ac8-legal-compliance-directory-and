WordPress plugin creating searchable directories of legal compliance professionals, consulting firms, and regulatory service providers. Features filterable directory listings by compliance framework (GDPR, HIPAA, SOC 2, ISO 27001), jurisdiction, service type, and industry vertical. Each listing includes consultant profiles, certification badges, case study summaries, client reviews, and contact forms. Supports premium listing tiers with featured placement, backlink do-follow, and lead notification emails. Admin dashboard shows directory analytics, top-searched compliance terms, and listing conversion rates. Includes schema markup for rich snippets, XML sitemap integration, and automated listing verification reminders. PHP 8.1+, WordPress 6.x+.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate via WordPress admin > Plugins
3. Configure via Settings > Legal Compliance Directory and Listings Plugin
